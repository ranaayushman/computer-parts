The CPU (Central Processing Unit)

The CPU, also known as the brain of the computer, is responsible for carrying out instructions and performing calculations. It processes input and produces output.

Key Facts About the CPU

  • Central Role – The CPU processes all instructions from programs and applications.
  • Speed – The faster the CPU, the more quickly tasks are completed.
  • Components – Includes ALU (Arithmetic Logic Unit) and Control Unit.
  • Location – Located on the motherboard inside the computer case.
